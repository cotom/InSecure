### Installation

1. Install nvm using brew and install latest version of Nodejs
``` bash
brew install nvm
nvm install --lts
```

2. Clone Repo
``` bash
git clone https://github.com/cotom/InSecure.git && cd InSecure
```

3. Install and run
``` bash
npm install && npm start
```
